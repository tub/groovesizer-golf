Groovesizer Golf
------------------------

In the Arduino IDE, open the file named "Groovesizer_Golf_" followed by the version number. All dependent files will be opened and accessible in tabs.

To compile the firmware, you will need to have the libraries in the "Required Libraries" folder installed. Generally, you should be able to just drag and drop the libraries in you Arduino Libraries folder, but for more help on installing libraries, visit http://arduino.cc/en/Guide/Libraries
